Describe the concept of Hierarchical Database.

## Asistente
Hierarchical database is a type of database structure in which data is stored and organized in a hierarchical structure. This structure is similar to a tree structure, with a single root node as the topmost node of the tree, subsequent levels of data branching off from the root node. Each node in the tree can contain one or more data items, and data items can be linked together by logical relationships, such as parent-child relationships. In a hierarchical database, information is accessed using a unique path, making the data easier to access and manage.
