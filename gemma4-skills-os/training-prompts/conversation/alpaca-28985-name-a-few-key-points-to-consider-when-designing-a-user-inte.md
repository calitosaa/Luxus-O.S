## Usuario
Name a few key points to consider when designing a user interface.

## Asistente
When designing a user interface, some key points to consider include usability, accessibility, aesthetics, responsiveness, responsiveness, readability, and consistency. Usability refers to how easy it is for a user to find what they need and understand how to use the interface. Accessibility refers to making the interface available for different types of disabilities. Aesthetics refers to how visually attractive the interface is. Responsiveness refers to how quickly the interface responds to user input. Readability refers to how easily the user can interpret written text. Finally, consistency refers to how similar the user experience between different parts of the interface is.
