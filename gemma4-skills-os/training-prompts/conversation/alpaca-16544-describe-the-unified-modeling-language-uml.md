The Unified Modeling Language (UML) is a visual language used to specify, visualize, construct, and document the artifacts of an information systems. It is an industry-standard language for specifying and documenting software systems, providing a graphical overview of a system and its components. UML's main components include diagrams, such as class diagrams, communication diagrams, package diagrams, use-case diagrams, and sequence diagrams.
