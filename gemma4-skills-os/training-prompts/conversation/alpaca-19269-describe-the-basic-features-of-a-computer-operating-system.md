## Asistente
A computer operating system is a software program that runs on the computer hardware and allows for user interaction and system management. It manages the hardware and acts as an interface between the user and the computer hardware. An operating system facilitates basic tasks such as controlling the input/output devices, starting programs, memory management, task scheduling, multitasking and protection of data on the computer.
