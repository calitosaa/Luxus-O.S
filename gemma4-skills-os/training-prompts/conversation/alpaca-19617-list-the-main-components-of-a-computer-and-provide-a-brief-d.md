## Asistente
The main components of a computer are the Central Processing Unit (CPU); Memory; Storage;Input/Output (I/O); and the Motherboard. The CPU processes data and executes commands. Memory stores programs and data that the CPU can access. Storage holds data so that the computer can access and use it later. Input/Output devices allow data to be sent to the CPU and receive data from the CPU. The Motherboard houses all the components of the computer and allows them to communicate with each other.
