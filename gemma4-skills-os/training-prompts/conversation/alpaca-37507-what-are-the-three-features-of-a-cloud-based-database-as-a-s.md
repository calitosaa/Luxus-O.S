The three main features of a cloud-based DBaaS are scalability, cost efficiency, and backups. Scalability allows you to increase or decrease the resources used by the DBaaS with ease. DBaaS is cost efficient because you only pay for the resources you need, when you need them. Finally, backups enable you to restore the database if necessary.
