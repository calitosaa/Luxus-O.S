## Asistente
Blockchain technology is a type of distributed ledger technology (DLT) which records digital data in an immutable and secure way. The data, which can consist of cryptocurrency transactions, contracts, or any other information, is stored in a decentralized network and consists of individual “blocks” which are linked and secured through cryptography. This makes it impossible to alter data without detect, as each block is verified by the network before being added to the “chain”.
