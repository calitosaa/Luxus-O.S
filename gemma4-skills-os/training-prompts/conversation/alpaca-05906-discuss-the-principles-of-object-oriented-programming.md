## Asistente
Object-oriented programming (OOP) is a programming paradigm based on the concept of "objects", which may contain data fields, and code, in the form of procedures known as methods. The principles of OOP are encapsulation, abstraction, polymorphism, and inheritance. Encapsulation is the ability to combine methods and data into a single unit, abstraction is the ability to hide the details of an implementation and only show the essential features, polymorphism is the use of a single interface for multiple data types, and inheritance is the ability for one class to extend the functionality of another.
