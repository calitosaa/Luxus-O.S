## Asistente
Objective: The goal of the game is to be the first player to reach a set number of points. 

Game Play: 
1. Each player shows their hand with one of three gestures – Rock, Paper or Scissors. 
2. Rock beats Scissors, Scissors beats Paper, and Paper beats Rock. 
3. If both players show the same gesture, then it is a draw and both players start again. 
4. The winner of each round earns one point. 
5. The first player to reach the set amount of points is the winner.
