## Asistente
Supervised learning and unsupervised learning are two main approaches to machine learning.  Supervised learning requires labeled data and consists of algorithms that can learn to make predictions from the existing data set. In contrast, unsupervised learning does not require labeled data and consists of algorithms that can find patterns and insights without human intervention.
