## Asistente
Conditional statements are statements in programming languages that allow the programmer to execute different code depending on the result of a given condition. A conditional statement consists of a condition, such as a Boolean expression, followed by a statement or block of code that is executed if the condition is true. If the condition is false, either an alternative statement or block of code can be executed.
