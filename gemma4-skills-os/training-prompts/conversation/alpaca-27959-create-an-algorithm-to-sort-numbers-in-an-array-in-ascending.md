## Asistente
The algorithm to sort numbers in an array in ascending order is: 
1. Set a marker for the first element in the array. 
2. Compare the marker to every other element in the array. 
3. Swap the positions of the marker and the element if the element is smaller than the marker. 
4. Set the marker to the next element and repeat steps 2 and 3 until the end of the array is reached. 
5. Repeat steps 1-4 until all elements in the array are sorted in ascending order.
