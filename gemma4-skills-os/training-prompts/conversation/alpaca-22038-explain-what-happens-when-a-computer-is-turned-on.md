## Asistente
When a computer is turned on, the operating system is loaded into the main memory. This process is known as booting, and it allows the computer to access all its resources and prepare it for user input and other tasks. After the operating system is loaded, the computer checks for any hardware and software connections, and it is then ready to process user commands.
