How do you create a linked list in C++?

## Asistente
Creating a linked list in C++ is a relatively simple process. The first step is to create a "Node" structure. The Node will contain the data which will be held in the linked list, in addition to the address which will point to the next Node in the list. To create the linked list, first a head pointer is declared and initialized to null, which will hold the address of the first Node in the list. The nodes are then created, and their next address is updated. Finally, the head pointer address is updated to the newly created Node and the linked list is complete.
