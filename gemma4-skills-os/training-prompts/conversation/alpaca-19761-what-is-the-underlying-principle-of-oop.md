## Asistente
The underlying principle of OOP is modularity, which makes it easier to maintain, extend, and adjust the program code. This is done by breaking the program code into discrete, distinct modules that represent objects, which are defined by their attributes and the operations that can be applied to them. This allows for code reuse and modularity through inheritance, which is when a new object can be created by using the characteristics of an existing object.
