## Usuario
Describe the concept of an Autoencoder.

## Asistente
An autoencoder is an artificial neural network used for unsupervised learning in which the input is the same as the output. It is used for data compression and dimensionality reduction by learning a compressed representation of the data in a lower dimensional space. The input is mapped to a lower-dimensional representation called an encoding by layers of neural networks, and then reconstructed into the original input by the network. The network learns the optimal encoding from the training data.
