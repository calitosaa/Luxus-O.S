## Asistente
The process of inserting a new node into a Linked List involves following steps: 
1. Create a new node.
2. Assign the data to the new node.
3. If the Linked List is empty, then assign the new node as the head node.
4. Otherwise, traverse the list until you find the final node, which is the last node of the linked list.
5. Make the new node's next reference the same as the last node's next reference. 
6. Make the last node's next reference be the new node.
