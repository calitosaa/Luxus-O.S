## Usuario
Describe the differences between BDD & TDD

## Asistente
Behavior Driven Development (BDD) focuses on the behavior of the system being developed, while Test Driven Development (TDD) focuses on the tests that must be written to verify the behavior. TDD involves writing a test first and then writing the code to pass it, while BDD involves writing a test feature first and then developing the code to make it pass.
