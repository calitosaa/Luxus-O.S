## Asistente
We can prove that 1+2+3+4+5=15 using mathematical induction. 

Base case: When n=1, 1+2+3+4+5=15 which is true. 

Induction hypothesis: Assume that 1+2+...+n = n(n+1)/2 and the equation holds true for some n. 

Induction step: We need to show that the equation 1+2+...+n + (n+1) = (n+1)(n+2)/2 is also true. 

Substituting the induction hypothesis: 1+2+...+n + (n+1) = n(n+1)/2 + (n+1) = n(n+1) + 2(n+1)/2 = (n+1)[n+2/2] = (n+1)(n+2)/2

Since the equation holds true for the base case and the induction step, for any natural number n, 1+2+3+4+5=15.
