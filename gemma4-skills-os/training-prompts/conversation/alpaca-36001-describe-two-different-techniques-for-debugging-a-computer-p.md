## Asistente
Two different techniques for debugging a computer program are using print statements and using a debugging tool. Using print statements involves inserting statements into the code that will print out the variable values at that point in the code, which can help identify where the problem is occurring. Another technique is to use a debugging tool, which can provide a detailed view of the code and make it much easier to identify the source of the problem. It can also provide options to step through the code line by line and analyze each line of code.
