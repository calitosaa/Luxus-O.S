## Asistente
A programming language is a set of instructions written in a specific syntax that can be interpreted and executed by a computer, while software is a program or collection of programs that are designed to perform specific tasks. Software generally manipulates data, creates graphs or diagrams, or enables interaction across a network, while programming languages are used to tell the computer how to interpret and execute the instructions. Programming languages are used to create software and many different languages can be used to create the same software.
