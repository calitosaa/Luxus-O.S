## Asistente
The data model should store information about users, restaurants, and reviews. The user table should store basic info about the users such as their username, email address, and other profile details. The restaurant table should store information about the restaurants such as name, address, category, and other details. The reviews table should store ratings and reviews given by the users for different restaurants. It should also store the user ID and the restaurant ID to keep track of which review belongs to which user and restaurant. Additionally, the data model should also store user preferences and recommendations, as well as different types of filters that can be used to narrow down the search results.
