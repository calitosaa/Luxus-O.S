## Usuario
Describe how the genetic code is read and translated in the synthesis of a protein?

## Asistente
The genetic code is first read in mRNA codons, each consisting of three nucleotides. The mRNA codons are then translated into amino acids which are linked together with peptide bonds to form a polypeptide chain. During translation, transfer RNA molecules carry the appropriate amino acids and line them up in the correct order. Once the full polypeptide chain is formed, the protein folds into its three-dimensional structure and is ready to be used by cells.
