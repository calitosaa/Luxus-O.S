## Asistente
Examples of bias in machine learning models include data bias, language bias, and algorithmic bias. Data bias occurs when the dataset used to train the model contains information that is biased, such as having more male examples or favouring certain ethnic groups. Language bias can manifest in the lack of appropriate vocabularies or syntactic suggestions for certain demographics. Algorithmic bias occurs when the algorithm itself is written in such a way that is biased, for example, when the algorithm treats females differently than males.
