## Usuario
Differentiate between abstract classes and interface

## Asistente
An abstract class is a class that contains one or more abstract methods which are methods without an implementation. An interface is a type that defines a set of behaviors that a class must implement. An interface does not contain any implementation, while an abstract class may contain implementation. An interface can be implemented by multiple or any class, while an abstract class can only be inherited by a single class.
