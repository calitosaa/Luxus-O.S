## Asistente
A GPS navigation system uses satellites to detect its location. The GPS receiver sends signals to the satellite which then send signals back containing information about the receiver's location, speed, and direction. This data is then used to calculate the distance to other points, enabling the user to receive directions and a map with the location of their destination.
