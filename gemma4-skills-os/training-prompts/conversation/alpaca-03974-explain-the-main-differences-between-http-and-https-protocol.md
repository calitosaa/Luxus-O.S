## Usuario
Explain the main differences between HTTP and HTTPS protocols.

## Asistente
HTTP (Hypertext Transfer Protocol) is a protocol used for transferring files and data between a client (web browser) and a server. HTTPS (Hypertext Transfer Protocol Secure) is an extension of HTTP and uses SSL/TLS (Secure Sockets Layer/Transport Layer Security) to encrypt communication between a client and a server. This encryption provides privacy and data integrity between two communicating applications. Additionally, HTTPS requires an SSL or TLS certificate to be installed on the website while HTTP does not.
