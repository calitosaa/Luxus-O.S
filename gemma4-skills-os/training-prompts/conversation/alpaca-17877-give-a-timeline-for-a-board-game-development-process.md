## Usuario
Give a timeline for a board game development process.

## Asistente
The timeline for a board game development process can be divided into six stages: idea exploration, game design, playtesting, graphic design, prototype production, and distribution.

Idea Exploration: In this stage, the game developer brainstorms ideas, develops the game's core mechanic, and tests its viability.

Game Design: Here, the developer refines the rulebook, game balance, and game flow.

Playtesting: Playtests are conducted with focus groups to find out which aspects of the game are fun and which require improvement.

Graphic Design: At this stage, artwork, illustrations, and card layouts, are created.

Prototype Production: The all components are produced and out together, creating the physical and digital prototypes.

Distribution: Finally, the game can be put on the market!
