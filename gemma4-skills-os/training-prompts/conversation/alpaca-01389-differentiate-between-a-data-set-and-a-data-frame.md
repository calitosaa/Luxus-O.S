## Asistente
A data set is a collection of data points that are organized into tables or spreadsheets. A data frame is a two-dimensional data structure with labeled axes where each axis is a variable and each row consists of a set of related values. Data frames can contain columns of different data types and are often used to manipulate data sets in the R language.
