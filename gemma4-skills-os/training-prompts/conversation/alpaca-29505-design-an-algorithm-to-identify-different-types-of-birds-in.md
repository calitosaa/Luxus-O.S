Contexto/Input:

## Asistente
The algorithm should first identify the bird in the image using an object detection model. After that, the model should use image similarity-based methods to identify the type of bird in the image. This can be done by comparing features such as size, shape and color of the bird, as well as texture of its feathers. Finally, the algorithm should use existent labeled data to determine which type of bird is in the image.
