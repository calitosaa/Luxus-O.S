## Asistente
A fixed-width font is a type of font where each character takes up the same amount of space, while a proportional font is a type of font where each character takes up an amount of space based on its size. Fixed-width fonts are typically used to display text in a structured way, such as when coding, while proportional fonts are mainly used for text display, such as when formatting a document.
