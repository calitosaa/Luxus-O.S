What is the difference between Responsive Design and Adaptive Design for webpages?

## Asistente
The difference between Responsive Design and Adaptive Design for webpages is that Responsive Design is a website design approach that uses flexible layouts and media queries to create a webpage that displays well and is usable across multiple devices. Adaptive Design uses predetermined breakpoints or predetermined screen sizes that the webpage is designed to adapt to, depending on the device.
