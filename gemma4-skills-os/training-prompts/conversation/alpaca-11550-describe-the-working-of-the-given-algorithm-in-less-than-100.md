## Asistente
K-Means Clustering is an unsupervised machine learning technique used to find clusters or groups of similar data points within a dataset. The algorithm first randomly initializes ‘k’ number of centroids, then proceeds to assign each data point to the closest cluster by calculating the distance between them. Then, the algorithm updates the centroids to the average of the data points within that cluster before repeating the process again until the centroids don’t move anymore. This iterative process is then used to group the data points based on the similarities within them and assign them to the corresponding clusters.
