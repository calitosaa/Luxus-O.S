## Asistente
The main difference between a firewall and an antivirus system is that firewall focuses on control of incoming and outgoing traffic, while an antivirus system focuses on detection and prevention of malicious software. Firewalls are typically implemented as a network security device, while antivirus software is installed on machines to protect against malicious code.
