## Asistente
Algorithm to check if a given string is a palindrome:

1. Take the string as input.
2. Initialize two pointers at the start and the end of the string.
3. Compare the characters at the two pointers.
4. If the characters are the same, move the two pointers towards the middle of the string.
5. Repeat step 3 until either the two pointers reaches the middle or the characters are different.
6. If the two pointers reach the middle of the string, the string is a palindrome. Else, it is not a palindrome.
