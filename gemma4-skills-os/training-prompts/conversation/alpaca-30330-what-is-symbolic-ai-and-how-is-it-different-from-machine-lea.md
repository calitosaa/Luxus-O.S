## Usuario
What is symbolic AI and how is it different from machine learning?

## Asistente
Symbolic AI is the practice of using symbolic rules and representations, or symbolic logic, to program computers to complete tasks. A symbolic AI system is designed to replicate human-like reasoning and problem-solving abilities. Unlike machine learning, which relies on statistical methods to act on data, symbolic AI systems use predefined rules and logic in order to solve problems. 

Machine learning can be thought of as a subset of symbolic AI, as it relies on predefined forms of data that are then acted on by a system. The key difference between the two is that while symbolic AI relies on pre-defined rules and representations, machine learning algorithms use training data to learn patterns and make decisions. As such, machine learning is considered to be more automated and adaptive than symbolic AI.
