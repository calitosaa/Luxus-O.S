Explain the concept of a fully connected network.

## Asistente
A fully connected network is a type of artificial neural network in which each neuron or processing unit is connected to every other neuron. In this type of network, each input feature is connected to every neuron in the hidden layer, and every hidden neuron is connected to each output neuron. This allows for a complex network of pathways for data to flow through, allowing for efficient data-processing and training.
