What is the purpose of the `else` clause in a `try-except-else` statement?

## Asistente
The `else` clause in a `try-except-else` statement is executed if the code in the `try` block executes successfully, and no exceptions are raised. This clause is used to provide a code block to perform certain operations that should only execute if the `try` block executes without raising any exceptions.
