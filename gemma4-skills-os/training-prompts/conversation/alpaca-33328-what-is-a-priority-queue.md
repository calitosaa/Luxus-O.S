What is a priority queue?

## Asistente
A priority queue is a data structure where items are stored in an order according to the priority assigned to them. It works by assigning a priority to each item that is added to the queue. The item with the highest priority is removed from the queue first, followed by the item with the next highest priority, and so on. This makes it useful for ordering tasks according to their importance.
