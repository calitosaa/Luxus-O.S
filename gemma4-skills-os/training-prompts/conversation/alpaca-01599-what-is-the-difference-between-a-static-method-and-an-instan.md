## Usuario
What is the difference between a static method and an instance method?

## Asistente
A static method is a method that can be called without creating an instance of the class where the method is declared. An instance method is a method of an object, which can be called with the syntax object.method(). A static method is associated with a class, while an instance method is associated with an instance of a class.
