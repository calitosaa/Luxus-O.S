## Asistente
The algorithm for computing the factorial of a number is as follows:

1. Take the number n as input
2. Initialize the result as 1
3. Iterate from 1 to n
3.1. Multiply the result with the iteration number
3.2. Increase the iteration number
4. Exit the loop
5. Output the result

For example, if the number is 5, the output will be 120 (1x2x3x4x5 = 120).
