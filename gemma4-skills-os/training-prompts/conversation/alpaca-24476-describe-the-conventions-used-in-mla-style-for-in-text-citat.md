## Asistente
According to MLA style, in-text citations should follow the author-page format. This means that when referring to a source within the text of a document, the author's last name and the page number of the source should be included. For example, if the author's name is Smith and the page number is ‘15’, then the in-text citation should appear as ‘(Smith 15)’. Additionally, when citing sources in the bibliography, the author's last name, title of the book, publication information, and page numbers should be listed.
