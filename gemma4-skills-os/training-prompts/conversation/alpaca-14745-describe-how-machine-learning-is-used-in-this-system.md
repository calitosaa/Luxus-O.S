## Asistente
Machine learning is used in this system to learn patterns in handwritten digits. It uses a training phase to identify features of the digits such as shape, weight, stroke and spacing in order to distinguish them from one another. During the testing phase, the ML model makes predictions on new hand-written digits by analyzing the same features with the previously trained ones.
