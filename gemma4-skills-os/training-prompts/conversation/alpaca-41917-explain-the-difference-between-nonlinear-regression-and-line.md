Explain the difference between nonlinear regression and linear regression.

## Asistente
The main difference between linear and nonlinear regression is that linear regression models assume a linear relationship between the input variables(x) and the target variable(y) while nonlinear regression models allow the relationship to be nonlinear. Linear regression models estimate linear coefficients between the variables that represent the relationships between the variables and can predict the target variable. Nonlinear regression models estimate nonlinear coefficients and are used to capture more complex relationships between the variables.
