## Usuario
Describe the kind of artificial neural network architecture that was used in the AlphaGo Zero algorithm.

## Asistente
The AlphaGo Zero algorithm uses a deep convolutional neural network (CNN) for the reinforcement learning architecture. The neural network consists of multiple layers of neurons, with each layer connected to the succeeding and preceding layer. The network is split into two parts, a policy network and a value network. The policy network is used to predict the probability of a given action and the value network is used to predict the expected outcome of a given move.
