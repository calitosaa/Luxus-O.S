## Asistente
Garbage in, garbage out (GIGO) is a saying which means that the quality of output is only as good as the quality of the input. It is commonly used when discussing computers and computing to mean that if the input data is wrong, the output of the program will also be wrong. This is because computers rely on the input data they receive to find the results, so if the input is incorrect then the output will also be incorrect.
 
GIGO is a reminder to always double check the data that is entered into the computer and make sure it is the correct data before running the program. It can also be used to explain why search engines are sometimes inaccurate. For example, if incorrect information is inputted into a search engine, then the results will also be incorrect. Therefore, it is important to check the accuracy of both inputs and outputs.
