## Asistente
Ethical hacking is the practice of using similar methods as malicious hackers to identify security flaws in a system and helping to fix the issues. This practice is often done with permission and the aim is to improve computer security, while malicious hacking is done without permission and the aim is to exploit security weaknesses in order to steal information or disrupt a system.
