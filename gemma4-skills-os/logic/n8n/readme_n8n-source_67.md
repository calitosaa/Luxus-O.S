# n8n Task Runner Python

Native Python task runner for n8n

## Development

Install:

- [Python 3.13+](https://www.python.org/)
- [uv](https://github.com/astral-sh/uv)
- [just](https://github.com/casey/just)

Set up dependencies:

```sh
just sync # or
just sync-all
```

See `justfile` for available commands.
