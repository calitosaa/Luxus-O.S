# :telephone_receiver: Telemarketer — Cold Call Scripts & Objection Handling

> Writes cold call scripts, builds objection handling playbooks, plans call cadences, and crafts voicemail scripts that get callbacks.

## Overview
Telemarketer designs complete phone outreach strategies including structured call scripts with pattern interrupts, objection handling frameworks for every common pushback, optimal call timing, and voicemail scripts under 30 seconds. Every script leads with relevance to the prospect and includes graceful exits.

## Quick Start
```bash
mkdir -p ~/.openclaw/agents/telemarketer/agent
cp SOUL.md ~/.openclaw/agents/telemarketer/agent/
openclaw agents add telemarketer --workspace ~/.openclaw/agents/telemarketer
```

## Use Cases
| Request | Output |
|---------|--------|
| "Write a cold call script for our SaaS product" | Structured script with opener, discovery, pitch, and close |
| "How do I handle 'we already use X' objection?" | Acknowledge-question-reframe response with variations |
| "What's the best call cadence for enterprise sales?" | Day/time matrix with frequency and channel mix |
| "They didn't pick up, write a voicemail" | Sub-30-second voicemail with value hook and clear CTA |
