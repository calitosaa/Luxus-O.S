# Luxus-O.S — Repositorio Centralizado de Agentes y Recursos

Colección organizada de repositorios clasificados por categoría.

## Estructura

### `skills/`
Skills y agentes para Claude Code y otros asistentes:
- `awesome-agent-skills` — VoltAgent skills collection
- `antigravity-awesome-skills` — Antigravity skills y plugins
- `ruflo` — ruvnet agent framework
- `superpowers` — obra/superpowers commands y skills
- `everything-claude-code` — Recursos completos para Claude Code
- `claude-mem` — Sistema de memoria persistente
- `browser-use` — Skill de navegación web autónoma
- `awesome-claude-skills` — ComposioHQ skills collection
- `agency-agents` — Agentes por industria/departamento

### `plugins/`
Plugins e integraciones con servicios externos:
- `stitch-mcp` — MCP para crear diseños en Stitch
- `notebooklm-py` — Conexión a NotebookLM para búsqueda de fuentes

### `workflows/`
Flujos de trabajo listos para usar y como base para crear nuevos:
- `n8n-workflows` — Workflows de n8n para automatización

### `systemprompt-leaks/`
Recopilaciones de supuestos system prompts filtrados de LLMs:
- `system_prompts_leaks` — Leaks de Anthropic, OpenAI, Google, xAI, Perplexity
- `leaked-system-prompts` — Colección adicional de prompts filtrados

### `entender-comportamiento/`
Proyectos para entender el comportamiento de agentes y sistemas:
- `n8n` — Código fuente completo de n8n (workflow automation)
- `openclaw` — Framework de comportamiento de agentes

### `diseño/`
Recursos de diseño, UI/UX y Material 3:
- `AionUi` — AionUI design system
- `material-3-expressive-catalog` — Catálogo Material 3 Expressive
- `beercss` — BeerCSS con Material 3
- `ui-ux-pro-max-skill` — Skill de diseño UI/UX Pro Max
