# Archivos que necesitan fuentes verificadas

Archivos con `needs_sources: true` en su frontmatter o con secciones `[NEEDS_SOURCE]`.

## Formato

```
- [ruta/al/archivo.md](../ruta/al/archivo.md) — sección sin fuente: "nombre de la sección"
```

*Este archivo se actualiza automáticamente cada vez que se crea o edita un archivo con `[NEEDS_SOURCE]`.*
